|                                 |
|     HackWars Database Setup     |

|         to run HackWars         |
| http://dev.mysql.com/downloads/ |
|                                 |
+---------------------------------+
|                                 |
| Config db-import before running |
|                                 |
|  MySQL Path, DB User, DB Pass   |
|                                 |
|     Windows: db-import.bat      |
|     OSX/Unix: db-import.sh      |
|                                 |
+---------------------------------+
|                                 |
| Config db-export before running |
|                                 |
|    MySQLDump Path, User, Pass   |
|                                 |
|    Windows: db-export.bat       |
|    OSX/Unix: db-export.sh       |
|                                 |
